# مشروع كتاب: البيت ميثاق ورحمة

هذا المجلد يحتوي مسودة كتاب عربية عن التعامل بين الزوجين، وحقوق كل طرف وواجباته، مع جعل الإسلام هو الأصل الحاكم واستفادة عملية من أشهر مدارس وكتب العلاقات الزوجية عالميا.

الملفات:

- `book.md`: المسودة النهائية للكتاب بصيغة Markdown.
- `outline.md`: هيكل الكتاب والمنهج التحريري.
- `sources.md`: مصادر الأفكار وروابطها وما تم استخلاصه منها.

تنبيه: المسودة تربوية وإرشادية، وليست فتوى فقهية مفصلة ولا بديلا عن استشارة أهل العلم أو المختصين عند النزاعات، أو العنف، أو الأمراض النفسية، أو القضايا القانونية.
