# مشروع كتاب: الثراء والرزق في الإسلام والقرآن

هذا المجلد يحتوي مسودة كتاب عربية أصلية عن الرزق والمال والثراء في التصور الإسلامي والقرآني.

## الملفات

- `book.md`: المخطوط الكامل الأولي للكتاب.
- `outline.md`: خطة الكتاب وهيكله التحريري.
- `sources.md`: مصادر البحث والكتب المحورية والروابط.

## ملاحظة

الكتاب تعليمي وتربوي، وليس فتوى فقهية تفصيلية ولا نصيحة مالية شخصية. مسائل الزكاة والمعاملات والاستثمار والديون تحتاج إلى مراجعة أهل العلم والاختصاص حسب واقع القارئ وبلده.

## حالة المشروع

نسخة كاملة أولى قابلة للتوسيع والتحرير. الخطوة التالية المناسبة: مراجعة شرعية، ثم تحرير لغوي، ثم تحويل المخطوط إلى DOCX أو PDF عند الحاجة.
